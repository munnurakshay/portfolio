WayBot Project

WayBot – Autonomous Sidewalk Delivery Robot

WayBot is an autonomous delivery robot designed to simplify last-mile delivery for businesses, campuses, and households. Built using Arduino Mega and integrated with GPS and compass modules, WayBot can navigate to a given location with high accuracy while avoiding obstacles in real time. Equipped with ultrasonic sensors mounted on a servo motor, it actively scans its surroundings, detects obstacles, and chooses the safest path to continue its journey. The GPS module combined with the HMC5883L compass ensures accurate heading and path correction during travel. A custom mobile app connected via Bluetooth allows users to set delivery destinations, track progress, and control basic functions when needed. WayBot features intelligent delivery logic — it increases speed during travel, pauses for 30 seconds upon arrival, and automatically returns to its starting point. For smoother user experience, it supports QR-based delivery confirmation and app-based monitoring. By combining navigation, obstacle avoidance, and delivery management into one affordable solution, WayBot demonstrates how robotics can transform everyday delivery operations into efficient, autonomous processes.
Key Features
GPS-based autonomous navigation
Obstacle detection and avoidance using ultrasonic sensors
Compass-based heading correction
Mobile app with Bluetooth control & map integration
Smart delivery logic (pause, return, confirmations)
Affordable, DIY-friendly design for small-scale delivery

Components Used


Arduino Mega

NEO-6M GPS Module

HMC5883L Compass Module

L298N Motor Driver

HC-SR04 Ultrasonic Sensor

SG90 Servo Motor

HC-06 Bluetooth Module

DC Motors with Wheels

Battery Pack
